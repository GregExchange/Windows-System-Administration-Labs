## Objectives

## Environment
- OS: Windows (VirtualBox VM)
- Network Mode: NAT

## Scenario
User reports inability to access a required folder needed for daily work.

## Tools Used
- File Explorer
- Local Users and Groups
- NTFS Permissions
- Event Viewer

## Troubleshooting Process
- Gathered user information and error description
- Verified user account and group membership
- Checked NTFS permissions on target folder
- Identified misconfigured access control
- Applied corrective permissions
- Verified resolution with user

## Issue Identified
Incorrect NTFS permissions preventing authorized access.

## Resolution
Updated folder permissions to allow appropriate access based on group membership.
