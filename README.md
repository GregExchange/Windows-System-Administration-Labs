# Windows System Administration Labs

## Overview
This repository documents a 14-day hands-on lab series completed in a
Windows VirtualBox environment. The labs simulate real-world IT Support
and Junior System Administration responsibilities.

## Environment
- Host OS: Windows 11
- Guest OS: Windows 10/11 (VirtualBox)
- Network Mode: NAT
- Tools Used:
  - PowerShell
  - Event Viewer
  - Task Manager
  - Resource Monitor
  - Disk Management

## Skills Demonstrated
- Windows system administration
- Network configuration and troubleshooting
- User and group management
- NTFS permissions and access control
- Software installation and application support
- Performance diagnostics and OS repair
- Patch management and change control
- Backup and recovery
- End-user support and documentation

## Lab Structure
Each folder represents a daily lab with:
- Step-by-step procedures
- Commands used
- Screenshots (where applicable)
- Troubleshooting notes
- Documentation artifacts

## Purpose
These labs demonstrate practical, job-ready skills aligned with
entry-level IT Support and Junior System Administrator roles.
